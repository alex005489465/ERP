## 動作
### 文件結構
```
com.company.project/
├── Application.java           ← 主啟動類
├── config/                   ← 配置類
├── controller/               ← 控制器層
├── service/                  ← 業務邏輯層
│   └── impl/                ← 服務實現類
├── repository/               ← 數據存取層
├── entity/                   ← 實體類
├── dto/                      ← 數據傳輸對象
│   ├── request/
│   └── response/
├── exception/                ← 異常處理
└── util/                     ← 工具類
```

### 其餘功能
- 增刪改item
- 庫存操作
  - 查stock
  - 決定要操作stock
  - 操作有入庫、出庫、轉庫、凍結、報廢...
    - 本質上都是入庫出庫，只是位置不同
    - 可用model代表
    - 出入庫->同名
    - 轉庫->一筆出庫、一筆入庫
    - 凍結、報廢->一筆出庫、一筆入庫(指定的地點，分別是一個地點)
